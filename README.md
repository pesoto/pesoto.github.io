# Personal Website

This is my personal website hosted on GitHub Pages. The site is built using HTML, CSS, and JavaScript.

## Structure
- `index.html` - Main page
- `styles.css` - Main stylesheet
- `script.js` - JavaScript functionality
- `profile_picture.jpg` - Profile image

## Development
The site is built with vanilla HTML/CSS/JS and hosted on GitHub Pages. No build process is required.

## Deployment
The site is automatically deployed through GitHub Pages when changes are pushed to the master branch.
